# Streuobstwiese

Die Gemeinde verpachtet der Dorfpflege seit den 80er Jahren eine Fläche nahe der Schienen im Dorf. Die Dorfpflege hat dort einen Obstgarten angelegt, der von den Mitgliedern gepflegt wird. Die Ernte wird von den Mitgliedern selbst verarbeitet.
Die Dorfpflege hat dort auch einen Schaukasten aufgestellt, in dem sie über ihre Aktivitäten informiert.
